Program Specifications

Simple Text - Jotter

Spell checker, grammar checker, (read aloud functions), auto fields

Word Processor - Pages

Lists, styles, HTML5, custom font browser, SVG fonts, Photos, Tables, CSS properties, Boxes, Columns, (and what simple text has).

Printer of HTML5 - Web Print

Margins, size & orientation, postscript or PDF export, No of pages, Header and footer (automatic), gutters, pamphlets.

Accounting/Budgets - Ledger

Auto formats, Common budgetting labels, automatic dates function, auto tabs, snazzy graphs, goal seek, online banking data compatible, common functions (sum, avg,loan,interest,bills).

Formatted Tables - Grids

Print area preview, HTML5 table (or XML), CSS header and footer, CSS boxes (eg. legend), No graphs, conditional formatting, borders, padding, backgrounds CSS3.

Maths from Tables - Abacus

Precision formatting, e etc. maths formulas. Custom defined functions, other maths functions. Auto formatting, better tab / tab group management. Complex charts including 3-4D. Vector and matrix support. Formula help. Science formulas (chemistry, physics, non-linear). Pivot tables (alternate groups of information).

Social Media - Social Net

Sends little posts to all sites at once and saves them. Manage security and privacy settings. Read all updates at once. Alert for user/keyword/blocks other things etc.

Blogging - Blogs

A simple text editor to create them. For all blogs, or simple site updates. Change look and feel of text. Insert photos through Flickr. Post YouTube videos. Saves blogs.

Web Pages and Presentation - Vision

Examples of layouts. Master layout. Lists, photos, photo effects, animation, other JQuery wizards. Presentation setting. Test in browser button. Marketing, SEO tools. Accessibility friendly. Flexible layouts.

Sites and Links - Hyper Help

Visual site linker, auto nav bars, automatic site map, per page keywords. How to host a website help. Submitting to search engines help.

Programming - Polyglot IDE

Web form setup, database setup. Javascript onclick methods (common uses). Form validation wizards. Office program designer (simple,API documentation, macros).

Serious Writing and Presentation Handouts - Scrolls

Biblography, fact checker, number tables. Simple plotted graphs, flow charts/UML etc. Print settings, save to PDF. Custom XML fields, XML tags (XLST and CSS support). Pictures links, hyperlinks, references to other parts of the document. Quotes, citations, custom fonts, scientific notation.

Contextual spell check and grammar check. International characters. Thesaurus, name generator, outline view. eBook publishing. Concordance, collaberation tools, editing tools. Better punctuation (auto checker). Advanced writers word count
