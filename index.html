Benefits

  • Non-cloud version that runs on a server at a company, therefore keeping all the data in-house.
  • Any files apart from hosted databases are saved to the users computer, therefore keeping the data private.
  • The modular small pieces of web applications e.g. buttons, menus are kept in the Sandbox, therefore programs are more easily constructed.
  • Easy to create new bespoke programs without requiring a computer science degree.
  • Low hardware and software requirements for any web applications made.
  • Applications can be made for phones, tablets, or any other computer with a recent web browser.
  • SSL encryption and user management
  • Open document formats used, and XML formats for interoperability.

Main Features

Sandbox: An application programming interface with small pieces of web application functionality collected together. Also tools for working with data and databases, or file uploads and downloads. Includes a language for building web applications, called Swan.

Paradise Office: An office suite built in the language 'Swan', with numerous small applications instead of big, giant programs eg. Microsoft Word.
