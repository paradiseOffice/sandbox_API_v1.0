Naming Conventions

Icons are named camel case, such as the following examples in the table.

ExampleDescription
webPrint_defaultIn the folder 'program_icons', for Web Print, the default look for the icon.
webPrint_hoverSame as above, but a hover graphic (may be included in default too)
editCopyIn the actions command icons folder, for the editing function 'copy'.

The id and program variable names keep the same conventions too, so webPrint is the name also used in the programming.
